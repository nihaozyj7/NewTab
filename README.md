# Edge新标签页扩展 - 安装和使用指南

## 扩展概述
这是一个基于HTML/CSS/JS构建的Edge浏览器新标签页扩展，提供了一个美观实用的搜索界面，支持多搜索引擎切换、主题设置、个性化定制等功能。

## 文件结构
- `index.html` - 主页面文件，包含所有UI和功能
- `manifest.json` - 扩展配置文件
- `icon/` - 图标目录
  - `icon/icon16.png` - 16x16像素扩展图标
  - `icon/icon48.png` - 48x48像素扩展图标
  - `icon/icon128.png` - 128x128像素扩展图标

## 安装步骤

### 方法一：开发者模式安装（推荐）
1. 打开Microsoft Edge浏览器
2. 在地址栏输入 `edge://extensions/` 并按回车
3. 开启右上角的“开发者模式”开关
4. 点击“加载解压的扩展”按钮
5. 选择包含以下文件的文件夹：
   - index.html
   - manifest.json
   - icon16.png
   - icon48.png
   - icon128.png
6. 扩展安装完成，新标签页功能即可生效

### 方法二：打包后安装
1. 将所有文件打包成ZIP格式
2. 在Edge扩展页面，拖拽ZIP文件到页面进行安装

## 功能特性

### 搜索功能
- 支持多个搜索引擎（必应、谷歌、百度、搜狗等）
- 可以自定义添加新的搜索引擎
- 搜索结果可选择在当前标签页或新标签页打开

### 个性化设置
- 深色/浅色主题切换
- 搜索框宽度调整
- 圆角大小设置
- 字体大小调整
- 顶部间距调节

### 用户体验
- 页面加载后自动聚焦搜索框
- 支持回车键快速搜索
- 美观的UI设计和动画效果

### 技术特性
- 符合Content Security Policy (CSP) 标准
- 无内联脚本，所有JavaScript代码分离到外部文件
- 适用于现代浏览器扩展开发

### 视觉优化
- 搜索引擎名称和搜索按钮高度统一
- 垂直居中对齐，视觉效果更加协调
- 字体权重优化，确保元素间视觉平衡
- 圆角样式动态适配，根据设置实时调整

## 注意事项
1. 需要所有必需文件才能正常工作
2. 扩展需要存储权限以保存个性化设置
3. 更换搜索引擎时使用 `%d` 作为搜索词占位符
4. 启动浏览器时，地址栏会显示扩展的 URL（如 `extension://.../index.html`），这是浏览器的正常行为，出于安全考虑无法隐藏。一旦开始搜索，地址栏通常会被自动隐藏或替换。

## 故障排除
如果扩展无法正常工作，请检查：
1. 所有文件是否在同一目录下
2. manifest.json文件格式是否正确
3. 图标文件是否存在且格式正确
4. 是否开启了开发者模式
