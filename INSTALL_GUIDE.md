# Edge新标签页扩展安装指南

## 准备工作

在安装扩展之前，请确保已完成以下准备工作：

1. 确保所有文件都在同一目录中：
   - index.html
   - manifest.json
   - icon/ 目录及其中的图标文件

2. 如果还没有图标文件，请运行以下命令生成：
   ```
   pip install Pillow
   python generate_icons.py
   ```

## 安装步骤

### 步骤1：启用开发者模式
1. 打开Microsoft Edge浏览器
2. 在地址栏输入 `edge://extensions/` 并按回车
3. 在页面右上角找到"开发者模式"开关并开启它

### 步骤2：加载扩展
1. 在扩展页面点击"加载解压的扩展"按钮
2. 浏览到包含扩展文件的文件夹（包含index.html和manifest.json的文件夹）
3. 选择该文件夹并点击"选择文件夹"

### 步骤3：验证安装
1. 扩展会出现在扩展列表中
2. 关闭当前标签页，打开一个新标签页
3. 您应该会看到新的自定义标签页界面

## 验证扩展功能

安装完成后，请验证以下功能是否正常工作：

1. **搜索功能**：在搜索框中输入内容并点击搜索按钮或按回车键
2. **设置功能**：点击右上角的齿轮图标，查看设置面板是否正常打开
3. **主题切换**：在设置中切换深色/浅色模式，查看效果
4. **个性化设置**：调整搜索框宽度、圆角等设置，查看是否保存
5. **搜索引擎管理**：尝试添加、编辑或删除搜索引擎，确认功能正常

## 故障排除

如果遇到问题，请尝试以下解决方案：

### 扩展未生效
- 确认已在 `edge://extensions/` 中启用扩展
- 确认manifest.json文件格式正确
- 确认所有必需文件都存在于同一目录中

### 图标缺失错误
- 确保图标文件已正确生成并位于icon/目录中
- 确认图标文件名为icon16.png、icon48.png、icon128.png

### 设置无法保存
- 确认扩展具有存储权限
- 检查浏览器控制台是否有JavaScript错误

### 搜索功能异常
- 确认搜索引擎URL格式正确
- 检查浏览器是否阻止了弹出窗口（影响新标签页打开）

## 自定义选项

### 更换图标
如果您想使用自己的图标：
1. 替换icon/目录中的图标文件
2. 确保文件名保持不变（icon16.png、icon48.png、icon128.png）
3. 重新加载扩展

### 修改功能
要修改扩展功能，只需编辑index.html文件：
- HTML部分控制界面布局
- CSS部分控制样式和外观
- JavaScript部分控制功能逻辑

## 卸载扩展

如需卸载扩展：
1. 访问 `edge://extensions/`
2. 找到"自定义新标签页"扩展
3. 点击"移除"按钮

## 技术支持

如果遇到无法解决的问题，请检查：
- Edge浏览器版本是否为最新版
- 扩展文件是否完整无损坏
- 是否有其他扩展冲突

扩展完全基于本地存储，所有设置都保存在您的设备上，不会上传到任何服务器。
